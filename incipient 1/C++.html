C++ Introduction

C++ is a widely-used programming language that builds on the foundation of C. It was introduced by Bjarne Stroustrup in the 1980s to support both procedural and object-oriented programming. Known for its speed and control over system resources, C++ is often chosen for applications where performance really matters—like operating systems, game engines, and embedded systems. Its ability to combine low-level programming with high-level abstraction makes it a versatile tool for developers.

To start C++, you need two things:
1.A text editor, like Notepad, to write C++.
2.A compiler, like GCC, to translate the C++ code into a language that the computer will understand.

Syntax

Example Code-
#include <iostream>
using namespace std;

int main()
{
  cout << "Hello World!";
  return 0;
}
Explanation-

#include <iostream>:
This line includes the input-output stream library so we can use functions like cout to print output.

using namespace std; :
This allows us to use standard library functions (like cout) without writing std:: every time.

int main() :
This is the starting point of every C++ program. The code inside this function will run when the program starts.

cout << "Hello World!"; :
This prints the text Hello World! on the screen using the standard output stream.

return 0; :
This ends the main function and returns the value 0 to indicate successful execution of the program.

Statements:

In a programming language, these programming instructions are called statements.It is important that you end the statement with a semicolon ";" .
If you forget the semicolon (;), an error will occur and the program will not run.

Comment

Comments are used to explain the code, making it easier to understand for others or for yourself when you revisit it later. They are ignored by the compiler, meaning they don’t affect how the program runs.

Types of comment:

1.Single-line Comment:

  • Starts with //
  • Used for short note or explanatin

    • 2.Multi-line Comment:

  • Start with /* and ends with */
  • Used for longer explanations or disabling blocks of code.

    • Example-1:

    // This is comment

      cout << "Hello World!";
      return 0;
    }
    Example-2:

    /* This is
    Multiple line
    comment */

      cout << "Hello World!";
      return 0;
    }

    Variable & Data Type

    Variable:

    A variable is like a container in memory where you can store some data, give it a name, and use it later in your program. In C++, you must always declare a variable before using it, which means telling the program what type of data it will hold.

    Example-
    int age =20;

    Here-

  • int - is the data type, which means the variable will store an integer.
  • age - is the name of the variable.
  • 20 - is the value assigned to variable age.

    • Data Type:

    Data types define what kind of data a variable can store. When you create a variable, you need to tell the compiler what type of data it will hold—like numbers, characters, or true/false values. This helps the program use memory efficiently and avoid errors.

    Common Data Types in C++:
  • int (2 or 4 bytes) – for whole numbers (like 5, -3, 100)
  • float(4 bytes)/ double (8 bytes)– for decimal numbers (like 3.14, -0.99)
  • char (1 byte) – for single characters (like 'A', 'z')
  • string (Not fixed) – for text (requires #include <string>)
  • bool (1 byte) – for true or false values
