Loop

Loops allow you to repeat a block of code multiple times as long as a certain condition remains true.

They’re especially useful because they help avoid repetition, minimize mistakes, and make programs easier to read and manage.


For Loop

A for loop is perfect when you already know how many times you want a piece of code to run. It keeps everything in one line—initialization, condition check, and update—making it short and neat.

Basic Structure
for (initialization; condition; update) {
// code to run repeatedly }
Example -
for (int i = 1; i <= 5; i++) {
cout << i << " ";
}

Nested Loops

Loop inside another loop is called a nested loop.

Example -
int main() {
for (int i = 1; i <= 3; i++) {
// Outer loop for (int j = 1; j <= 5; j++) {
// Inner loop cout << "* ";
}
cout << endl; // Moves to the next line after each row
}
return 0;
}

While Loop

Keep Running... As Long As the Condition Is True.

The while loop is used when you don’t know exactly how many times the code needs to run. It simply checks the condition before every iteration and runs the block only if the condition is true.

Basic Structure
while (condition) {
// code to repeat
}
Example -
int count = 1;

while (count <= 5) {
cout << count << " ";
count++;
}

do while Loop

Runs At Least Once—No Matter What!

Do while is a variant of the while loop.The do-while loop is a bit different from other loops. It always executes the code block first, and checks the condition afterward. So, even if the condition is false from the start, the code inside will run at least once.

Basic Structure
do {
// code to execute
} while (condition);
Example -
int i = 1;

do {
cout << i << " ";
i++;
} while (i <= 5);
