Function

A function is a block of code that performs a specific task. Instead of writing the same code multiple times, you can define it once as a function and simply "call" it whenever needed.

C++ provides some pre-defined functions, such as main(), which is used to execute code.

Types of Function -
1.Built-in: Already provided by C++ (e.g., cout, sqrt())
2.User-defined: Created by programmers to solve specific tasks

Basic Syntax:
return_type function_name(parameters) {
// code to execute
return value;
}
Example -
int add(int a, int b) {
return a + b;
}

Parameters and Arguments

When working with functions in C++, we often need to pass values in and out. That’s where parameters and arguments come in. They work together to help functions receive and use data.

What is a Parameter?
  • A parameter is a placeholder used inside a function definition.
  • It acts like a local variable that waits to receive a value when the function is called.
    • void greet(string name) { // 'name' is a parameter
    cout << "Hello, " << name;
    }
    What is an Argument?
  • An argument is the actual value you pass to a function when you call it.
    • greet("Ayesha"); // "Ayesha" is the argument

    Function Overloading

    With function overloading, multiple functions can have the same name with different parameters.

    Example -
    int myFunc(int x, int y) {
    return x + y;
    }
    double myFunc(double x, double y) {
    return x + y;
    }

    int main() {

    }

    Recursion in C++

    Recursion is a programming concept where a function calls itself to solve smaller parts of a bigger problem. It may sound tricky at first, but it’s actually a powerful technique used in problems like factorials, Fibonacci series, tree traversal, and more.

    How It Works:

    In recursion, a function keeps calling itself until it reaches a stopping point, known as the base case. Without a base case, the function would continue forever and crash the program.

    Basic Structure:
    void function() {
    // Base case (stopping condition)
    if (someCondition) return;

    // Recursive call
    function();
    }
    Example -
    int factorial(int n) {
    if (n == 0) return 1; // base case
    else return n * factorial(n - 1); // recursive step
    }

    Explanation -

    If you call factorial(5), it breaks down like this:
    5 * factorial(4)
    4 * factorial(3)
    3 * factorial(2)
    2 * factorial(1)
    1 * factorial(0)

    And finally returns 120.
