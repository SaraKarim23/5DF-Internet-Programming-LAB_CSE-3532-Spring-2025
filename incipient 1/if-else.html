If-Else

For conditional statements or instructions, we use If-Else.

Some logical condition fro mathematics -
  • Less than: a < b
  • Greater than: a > b
  • Less than or equal to: a <= b
  • Greater than or equal to: a >= b
  • Equal to a == b
  • Not Equal to: a != b

    • Basic Structure:
    if (condition) {
    // code runs if condition is true }
    else {
    // code runs if condition is false }
    Example-
    int age = 18;

    if (age >= 18)
    {
    cout << "You are an adult.";
    }
    else
    {
    cout << "You are underage.";
    }
    We Can Also Use else if for Multiple Conditions
    Example -

    if (score >= 90)
    { cout << "A grade"; }
    else if (score >= 80)
    { cout << "B grade"; }
    else
    { cout << "Keep trying!"; }



    SWitch-Case

    Switch-Case is an Alternative to Multiple If-Else Blocks.

    When you need to compare the same variable or expression against multiple values, writing too many if-else statements can get messy. Thatâ€™s where the switch-case structure makes things clean and organized.

    Basic Structure:
    switch (variable) {
    case value1:
    // code for value1
    break;
    case value2:
    // code for value2
    break;
    ...
    default:
    // code if no case matches
    }
    Example -
    int day = 3;
    switch (day) {
    case 1:
    cout << "Monday";
    break;
    case 2:
    cout << "Tuesday";
    break;
    case 3:
    cout << "Wednesday";
    break;
    default:
    cout << "Invalid day";
    }
    How It Works:
  • The value of day is checked.
  • If it matches case 3, the code prints "Wednesday"
  • The break stops it from running the next case.
  • If none match, default runs (like the else in if-else).
