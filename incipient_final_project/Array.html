Array

An array is like a row of boxes where each box can hold a value of the same type. Instead of declaring multiple variables for related data, you can use an array to store and access them easily using a single name with index numbers.

Declaring an Array:
data_type array_name[size];
Example -
int marks[5] = {90, 85, 78, 92, 88};

Here -

  • int is the type of data (integers)
  • marks is the array name
  • [5] means it can hold 5 elements

    • Accessing Elements:
    cout << marks[2]; // Output: 78
    Example using Loop:
    for (int i = 0; i < 5; i++) {
    cout << marks[i] << " ";
    }
    Size of an Array:

    To get the size of an array, you can use the sizeof() operator

    int Numbers[5] = {10, 20, 30, 40, 50};
    cout << sizeof(Numbers);

    Here Result: 20
