String

String is used to store and work with textâ€”like words, sentences, or even whole paragraphs. Unlike character arrays (char[]), the string type from the C++ Standard Library (#include <string>) is much more flexible and easier to use.

Example:
#include <iostream> #include <string>
using namespace std;
int main()
{
string name = "Ayesha";
cout << name;
return 0;
}

In the example above, we created a string named name that holds the value "Ayesha".

Key Features of C++ Strings:
  • You can easily check the length using .length() or .size()
  • Strings support many helpful functions like .substr(), .find(), .append(), etc.
  • You can combine strings using the + operator

    • String Concatenation

    The + operator can be used between strings to add them together to make a new string. This is called concatenation.

    Example:

    string first = "Hello";
    string second = "World";
    string result = first + " " + second;
    // result = "Hello World"

    String Length

    To get the length of a string, we use the length() function.

    Example:

    string x = "Hello";
    cout<<"Length of the string is:" << x.length();

    Acces of String

    You can access the characters in a string by referring to its index number inside square brackets [ ].

    Example:

    Access of first string-

    string x = "Hello";
    cout<< x[0];

    User input String
    Example:

    string first_name;
    cout << "first name: ";
    cin >> first_name;
    // get user input from the keyboard
    cout << "Your name is: " << first_name;

    // Type your first name: John
    // Your name is: John

    Using getline( ) function -

    string Full_Name;
    cout << "Full Name: ";
    getline (cin, Full_Name);
    cout << "Your name is: " << Full_Name;

    // Full Name: John Doe
    // Your name is: John Doe
